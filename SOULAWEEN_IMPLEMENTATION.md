# Soulaween 實作摘要

`games/soulaween/game.html` 已採用與其他六款相同的 `game-core.js`、`games.js` 與 `game-shell.css` 共用架構，包含：

- 4×4 雙面靈魂棋規則、相鄰翻轉、四連線收取、滿盤處理與先得 3 分。
- 人類、隨機與 MCTS 玩家。
- MCTS 勝率條；背景計算保留上一個值，終局固定顯示 100/0。
- 遊戲設定、難度／迭代數、開始新對局、Undo、日誌、規則與介紹分頁。
- 手機優先與桌面響應式介面。

舊的 `interface.html` 僅保留為網址相容轉址。蒐靈祭的完整回合、收取選擇與勝負規則已納入共用規則測試，避免日後修改共用介面時發生功能回歸。
